2. Создать и активировать виртуальное окружение, убедиться, что в нем нет пакета pillow
user@Treistan-MacBook-Air Example % python3 -m venv venv svetik_virt

.		..		svetik_virt	venv
user@Treistan-MacBook-Air Example % cd svetik_virt
user@Treistan-MacBook-Air svetik_virt % ls -a
.		..		bin		include		lib		pyvenv.cfg
user@Treistan-MacBook-Air svetik_virt % cd ..
user@Treistan-MacBook-Air Example % source svetik_virt/bin/activate
(svetik_virt) user@Treistan-MacBook-Air Example % pip freeze
(svetik_virt) user@Treistan-MacBook-Air Example %

3. Деактивировать виртуальное окружение
(svetik_virt) user@Treistan-MacBook-Air Example % deactivate
user@Treistan-MacBook-Air Example %
